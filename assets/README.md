# Audio Assets

## Background Music

Place your background music file here:

**Filename:** `interstellar-ambient-music_background-music.wav`

The application will automatically load and play this file on startup.

### Controls:
- **M key**: Toggle mute/unmute
- Music plays on infinite loop by default

### Supported Formats:
- WAV (recommended)
- OGG Vorbis
- MP3 (if SDL_mixer compiled with MP3 support)
- FLAC

### Notes:
- If the music file is missing, the application will continue without audio
- The file should be in the `assets/` directory relative to the executable
